# Bike Sales Analysis Dashboard
Interactive Excel dashboard analyzing customer demographics and purchasing behavior to identify factors influencing bike sales.

## Project Overview

This project analyzes customer demographic and purchasing data to identify patterns and factors influencing bike purchases. The analysis was performed using Microsoft Excel, Pivot Tables, Pivot Charts, and Interactive Slicers to create a business-focused dashboard.

## Objective

The objective of this project is to explore customer purchasing behavior and determine how factors such as income, age, gender, marital status, education, occupation, and commute distance impact bike purchase decisions.

## Dataset Information

The dataset contains customer-level information including:

- Customer ID
- Marital Status
- Gender
- Income
- Children
- Education
- Occupation
- Home Ownership
- Number of Cars
- Commute Distance
- Region
- Age
- Purchased Bike (Yes/No)

## Key Insights

- Customer income showed a strong relationship with bike purchase decisions.
- Certain age groups were more likely to purchase bikes than others.
- Commute distance influenced customer buying behavior.
- Demographic factors such as occupation and marital status impacted purchase patterns.
- Interactive filtering allowed deeper exploration of customer segments.
